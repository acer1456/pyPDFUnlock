# pyPDFUnlock

**一個給簡單的PDF解鎖小工具** 

本工具只是純粹個人方便使用，並嘗試做一個完整的git，故分享給需要的人。

### macOS
[Download](https://github.com/acer1456/pyPDFUnlock/releases/download/v0.1/pyPDFUnlock.MacOS.zip)

### Windows preview
[Not Yet](#)

## 🌼 目前的功能
* [x] 圖形化介面
* [x] 單、多個PDF選擇
* [x] 解開單、多個PDF
* [ ] 自定義檔案名稱（未來）
* [ ] 加上自定義密碼（未來）

> 更多其他正在開發中，也可以幫忙貢獻唷！

##  使用的套件
* tkinter
* ttkbootstrap
* pikepdf
* pyinstaller

# 如何使用
```console
$ pip install -r requirements.txt
```

```console
$ python main.py
```
